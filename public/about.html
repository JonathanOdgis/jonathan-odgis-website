About Jonathan Odgis

Jonathan Odgis is a recent honors graduate of the SUNY New Paltz School of Science and Engineering where he earned a Bachelor of Science degree in Computer Science. Jonathan's technical proficiencies include front-end web development languages and frameworks such as HTML5, CSS, Javascript, AngularJS, and JQuery. He is also competent in C# Scripting with the Unity Scripting API. Jonathan is proficient in working with the Unity Game Engine.

Jonathan completed an internship with a Sr. Software Developer in developing, maintaining, and documenting a Java Framework called the HObservable Pattern. This framework extends the functionality and flexibility of the Observer Pattern in Java allowing observers to be children of each other. This allows for applications with many components to communicate with each other without being tightly coupled to two observables at the same time.

Jonathan also received a Certification in Digital Design and Fabrication (3D Printing) from SUNY New Paltz. He is knowledgeable in 3D Modeling Software such as Autodesk Maya and Rhino3D. He developed a Unity Game utilizing Serial Input Communication using an Arduino Uno chip. He is experienced in working in Google Cardboard SDK in Unity as well.

In his free time is Jonathan is developing a video game using the Unity Game Engine and is working towards publicizing it by year-end.
